# Netflix Movies & TV Shows Insights

Interactive dashboard visualizing avg movies (10,389), TV shows (1.76), trends by year, top genres, ratings, country map.  
Screenshot: Netflix dashboard.png

## Key Insights
- Content growth peaks in recent years.
- US/Brazil lead production.
- TV shows: 62%, Movies: 38%.

## Tech Stack
- Power BI for visualization
- Kaggle dataset: [Netflix Movies and TV Shows: https://www.kaggle.com/datasets/shivamb/netflix-shows] 
- Skills: EDA, KPIs, geospatial analysis
